# Les Bonnes Pièces - Version finale

## Produits disponibles
- 5 pièces automobiles avec images
- Catégories : Optiques, Freinage, Essuie-glace

## Installation
1. Démarrer Apache et MySQL dans XAMPP
2. Importer `cleanup_5products.sql` dans phpMyAdmin
3. Accéder à http://localhost/les-bonnes-pieces/

## Technologies
- PHP 8.2 + MySQL
- JavaScript (fetch API)
- Bootstrap 5
