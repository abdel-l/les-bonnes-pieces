# Les Bonnes Pièces - E-commerce pièces automobiles

Projet BTS SIO SLAM — application web de vente de pièces automobiles.

## Prérequis

- XAMPP avec Apache et MySQL démarrés

## Installation

1. Importer la base de données `les_bonnes_pieces` dans phpMyAdmin (ou via MySQL)
2. Démarrer Apache dans XAMPP
3. Accéder à : http://localhost/les-bonnes-pieces/

## Technologies

- PHP (API backend)
- MySQL (base de données)
- JavaScript (frontend dynamique)
- Bootstrap (mise en page)

les-bonnes-pieces/
